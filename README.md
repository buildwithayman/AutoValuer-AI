# 🚘 AutoValuer AI – Car Price Prediction System

🔗 Live App:https://autovaluer.streamlit.app/

AutoValuer AI is a machine learning-powered web application that predicts car prices based on key vehicle specifications.  
Built with a clean, modern UI and deployed using Streamlit, this project is designed for real-world usability and portfolio showcase.

---

## 📌 Project Overview

This system uses a trained **Lasso Regression model** to estimate car prices by analyzing multiple vehicle features such as engine size, horsepower, mileage, and more.

The application provides:
- Real-time price prediction
- Confidence score (UI-based)
- Estimated price range
- Clean and professional user interface

---

## 🧠 Machine Learning Details

- **Algorithm Used:** Lasso Regression
- **Preprocessing:**
  - One-hot encoding for categorical features
  - Feature scaling using StandardScaler
- **Input Features:**
  - Symboling
  - Car Body
  - Drive Wheel
  - Engine Location
  - Wheelbase
  - Car Length
  - Car Width
  - Curb Weight
  - Engine Type
  - Cylinder Number
  - Engine Size
  - Horsepower
  - City MPG
